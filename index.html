About the Summit

Web Standards Summit 2026 focuses on practical techniques and emerging standards that help teams build fast, accessible, and future-friendly web experiences.

Key terms

HTML
HyperText Markup Language, the standard markup language used to define the structure and meaning of content on the web.
CSS
Cascading Style Sheets, the language used to control the presentation and visual layout of HTML documents.
Web Accessibility
The practice of designing and developing websites so that people with disabilities can perceive, understand, navigate, and interact with the web.

Featured Speakers

Alex Rivera

Alex is a standards advocate and chair of several working groups, helping browser vendors and developers collaborate on interoperable web features.

Topics
  • The State of Web Standards in 2026
  • Interoperability Across Modern Browsers
  • Building for the Next Decade of the Web

Priya Desai

Priya is a frontend architect focusing on semantic markup, maintainable design systems, and accessible component libraries for large-scale products.

Topics
  • Semantic HTML in Design Systems
  • Structuring Complex Interfaces Accessibly
  • Documentation-Driven Frontend Architecture

Fatima Khan

Fatima is an accessibility specialist who helps organizations integrate inclusive design into every stage of their product lifecycle.

Topics
  • Practical Web Accessibility Testing
  • Designing for Assistive Technologies
  • Accessibility as a Team Practice
